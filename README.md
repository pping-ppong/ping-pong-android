## 💬 App Description
```
팀플, 여행, 집안일, ... 다양한 모임의 할 일 어떻게 관리하고 계시나요?
서로 해야할 일을 파악하고 어려운 일은 다른 사람에게 넘겨요!
하나씩 주어진 일을 완료할 수록 펑하고 터지는 팝콘, 같이 모아봐요!
```

## 🛠️ Stack
* **Languange** : Kotlin
* **Library & Framewrok** : Android Studio, Retrofit2, Gilde, RxKotlin, Material
* **Model** : MVVM
